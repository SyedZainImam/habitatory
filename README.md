# Habitatory — From Celebrations to Creations

Premium event planning website built with Next.js, Tailwind CSS, and Sanity CMS.

## Quick Start

```bash
yarn install
yarn dev
```

- Website: http://localhost:3000
- Admin Panel: http://localhost:3000/studio
